# Slide Remote Web ユーザー向け使用ガイド

## Slide Remote Webとは

Slide Remote Webは、スマホをPowerPointのリモコンとして使うためのWebアプリです。

PCで発表セッションを作成し、スマホでQRコードを読み取ることで、スマホからスライドの「次へ」「戻る」などを操作できます。

## できること

- スマホからPowerPointのスライドを進める
- スマホから前のスライドに戻る
- スライドショーを開始する
- スライドショーを終了する
- 黒画面・白画面に切り替える
- QRコードでスマホを簡単に接続する
- 6桁コードで手動接続する

## 利用に必要なもの

- PowerPointを使うPC
- スマホ
- インターネット接続
- Googleアカウント
- PC側受信アプリ（receiver）

## 全体の流れ

```text
1. PCでSlide Remote Webにログイン
2. 発表セッションを作成
3. PC側receiverを起動
4. PowerPointをスライドショーモードにする
5. スマホでQRコードを読み取る
6. スマホからスライドを操作
```

## 1. PCでログインする

1. Slide Remote Webにアクセスします。
2. 「PCでログインして始める」または「ログイン」を押します。
3. Googleアカウントでログインします。
4. ダッシュボードが表示されます。

## 2. 発表セッションを作成する

1. ダッシュボードで「新しいセッションを作成」を押します。
2. セッション名を入力します。

例：

```text
卒論発表練習
ゼミ発表
研究発表本番
```

3. 「セッションを作成」を押します。
4. PC用セッション画面が表示されます。

## 3. PC側receiverを起動する

PowerPointを実際に操作するには、PC側でreceiverを起動する必要があります。

PC用セッション画面に表示されている6桁コードを確認します。

例：

```text
428193
```

ターミナルでreceiverフォルダに移動します。

```bash
cd receiver
```

次のコマンドでreceiverを起動します。

```bash
npm run dev -- 428193
```

`428193` の部分は、PC用セッション画面に表示されている6桁コードに置き換えてください。

起動に成功すると、ターミナルに次のような表示が出ます。

```text
Receiver started
Pairing code: 428193
Subscription status: SUBSCRIBED
```

このターミナルは閉じずに、そのままにしてください。

## 4. PowerPointをスライドショーモードにする

1. 操作したいPowerPointファイルを開きます。
2. スライドショーモードにします。
3. スライドショー画面を前面に表示します。

## 5. スマホでQRコードを読み取る

1. PC用セッション画面に表示されているQRコードをスマホで読み取ります。
2. スマホ用リモコン画面が開きます。
3. 画面上部にセッション名が表示されていれば接続成功です。

QRコードが読み取れない場合は、トップページの「スマホで接続する」から6桁コードを入力して接続できます。

## 6. スマホから操作する

スマホ用リモコン画面では、以下の操作ができます。

| ボタン | 操作 |
|---|---|
| 次へ | 次のスライドに進む |
| 戻る | 前のスライドに戻る |
| 発表開始 | スライドショーを開始する |
| 終了 | スライドショーを終了する |
| 黒画面 | 画面を黒くする |
| 白画面 | 画面を白くする |

## よくあるトラブル

### スマホ画面が404になる

スマホ用URLが間違っている可能性があります。  
PC用セッション画面のQRコードをもう一度読み取ってください。

また、アプリ側に以下のページが存在している必要があります。

```text
/remote/[sessionId]
```

### スマホでボタンを押してもPowerPointが動かない

以下を確認してください。

- receiverが起動しているか
- receiverのターミナルに `Subscription status: SUBSCRIBED` が出ているか
- PC用セッション画面の6桁コードとreceiver起動時のコードが同じか
- PowerPointがスライドショーモードになっているか
- PowerPointのスライドショー画面が前面にあるか

### receiverにログが出ない

以下を確認してください。

- commandsテーブルにデータが追加されているか
- receiverのSupabase設定が正しいか
- Supabase Realtimeが有効になっているか
- receiver起動後に新しい操作を送っているか

### Googleログインできない

以下を確認してください。

- Googleアカウントでログインしているか
- ブラウザのポップアップやCookieがブロックされていないか
- 管理者側のSupabase / Google OAuth設定が正しいか

## 終了方法

発表が終わったら、以下を行います。

1. スマホ画面で「終了」を押す
2. PowerPointのスライドショーを終了する
3. receiverを起動しているターミナルで `Ctrl + C` を押す
4. 必要に応じてWebアプリからログアウトする

## 注意事項

- PowerPointを操作するには、PC側receiverが起動している必要があります。
- Webアプリだけでは、PC上のPowerPointを直接操作できません。
- このアプリは、PC側receiverがキーボード入力を送ることでPowerPointを操作します。
- 発表中は、PowerPointのスライドショー画面を前面にして使用してください。
